# Gestion-Bal-Android
Application Android pour la gestion du bal du Lycée Pasquet

## Sommaire
- Installation
  - Pré-requis
  - Procédure

## Installation
### Pré-requis
- Android Studio
- Gestion-Bal-Windows
- MySQL/MariaDB

### Procédure
- Cloner le repos
- Importer le fichier database.sql dans votre serveur MySQL/MariaDB
- Ouvrir le dossier avec Android Studio
